# Condominio Río Caivico — Sitio estático

Entrega: sitio informativo estático siguiendo `instrucciones_sitio_rio_caivico.md` y `ajustes_visual_version_ecologica.md`.

Archivos principales:
- `index.html` — estructura semántica y contenido.
- `styles.css` — estilos (Flexbox + Grid), responsive y paleta verde.
- `script.js` — interactividad mínima (menú hamburguesa, sticky header, scroll suave).

Instrucciones rápidas:
1. Se han añadido imágenes de ejemplo (`maqueta.jpg` en la raíz y `assets/gallery1.jpg` ... `assets/gallery6.jpg`).
2. Si quieres reemplazar por fotos reales, sustituye esos archivos manteniendo los mismos nombres o actualiza las rutas en `index.html`.
3. Abre `index.html` en el navegador para previsualizar.

Notas importantes:
- Todas las imágenes de la galería usan `loading="lazy"` según requisito obligatorio.
- Proyecto usa solo HTML5, CSS3 y JavaScript vanilla.
- Evitar animaciones llamativas para mantener estética sobria.
