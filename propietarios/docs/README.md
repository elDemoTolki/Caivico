# Documentos del Condominio - Guía de Organización

## Descripción

Esta carpeta contiene los documentos administrativos, normativos y comunicados oficiales del Condominio Río Caivico. Los archivos se almacenan en formato **PDF** para garantizar compatibilidad, seguridad y facilidad de consulta.

## Tipos de Documentos Almacenados

### Documentos Normativos
- Reglamento interno del condominio
- Protocolos de convivencia y uso de espacios comunes
- Planos generales y normativas del condominio

### Documentos Administrativos
- Actas de reuniones de copropietarios
- Comunicados oficiales de la administración
- Calendario de mantenciones y trabajos programados
- Reportes financieros y administrativos

### Otros Documentos
- Avisos importantes
- Resoluciones administrativas
- Documentación de carácter general

## Estructura de Nombrado Recomendada

Para mantener una organización clara y consistente, se recomienda seguir los siguientes patrones de nomenclatura:

### Reglamentos y Normas
```
reglamento-interno.pdf
protocolo-convivencia.pdf
planos-generales.pdf
```

### Actas de Reuniones
```
acta-2026-03-15.pdf
acta-asamblea-2026-02-28.pdf
acta-comite-2026-01-20.pdf
```

### Comunicados
```
comunicado-2026-04.pdf
aviso-mantenciones-2026-03.pdf
resolucion-administrativa-2026.pdf
```

### Reportes Periódicos
```
reporte-financiero-2026-Q1.pdf
estado-proyectos-2026-marzo.pdf
```

## Directrices de Publicación

1. **Formato**: Todos los documentos deben estar en formato PDF
2. **Nomenclatura**: Usar guiones (-) para separar palabras, no espacios ni caracteres especiales
3. **Fechas**: Utilizar formato ISO (YYYY-MM-DD) para mayor claridad
4. **Versionado**: Si un documento se actualiza, puede incluirse versión: `documento-v2.pdf`
5. **Tamaño**: Se recomienda que los archivos no excedan 10 MB para descargas rápidas

## Procedimiento para Agregar Nuevos Documentos

1. Preparar el archivo en formato PDF
2. Nombrar el archivo siguiendo la estructura recomendada
3. Colocar el archivo en esta carpeta (`/propietarios/docs/`)
4. Actualizar el listado de documentos en el portal si es necesario
5. Notificar a los propietarios sobre la nueva documentación disponible
